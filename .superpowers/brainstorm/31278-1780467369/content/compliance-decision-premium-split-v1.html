Premium Split Workspace

Day la wireframe chi tiet cho phan Danh gia & Duyet. Muc tieu: dep hon, cao cap hon, nhung reviewer van cap quyen nhanh va ro ngu canh.

Desktop Wireframe
Desktop >= 1024px
Header: Product title | artist | status chips | last review
Left Context Rail
Current legal/review state
Suggested action card
Previous reject note / external legal note
Current approved permissions summary
Right Decision Workspace
Segmented choice: Pending / Approved / Rejected
If rejected/insufficient: reason editor appears inline
Permission toolbar: search, count, clear all, recommended
Permission cards grouped visually
Sticky selected permissions tray + primary save action
Mobile / Tablet Wireframe
Mobile < 640px / Tablet 640-1023px
Top summary card with title + compact status chips
Decision segmented control
Inline reason box when needed
Accordion: current context / previous note / current permissions
Permission picker as full-width stacked cards
Sticky bottom action bar: selected count + Save
Core Ideas
  • Ben trai chi de doc va tham chieu, ben phai danh cho thao tac.
  • Trang thai duoc chon bang segmented controls thay vi dropdown.
  • Permission cards duoc uu tien thi giac hon checkbox list thuan tuy.
  • Action footer luon hien de nguoi dung khong phai cuon tim nut luu.
Why It Fits This Case
  • Phan phap ly chi la context, khong chiem het su chu y.
  • Trong tam thuc su la cap quyen su dung san pham cho user.
  • Layout tao cam giac "workspace" cao cap hon form dialog thong thuong.
  • Hop voi desktop admin nhieu hon, nhung van co phuong an mobile ro rang.
