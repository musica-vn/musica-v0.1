Vue JSX best practices and differences from React JSX.

## Use When
- working in `.tsx` or JSX-based Vue render functions
- porting patterns from React JSX into Vue
- fixing attribute or event syntax that behaves differently in Vue JSX

## Do First
- confirm the task is actually JSX-based and not a normal SFC
- keep this skill scoped to JSX-specific differences instead of generic Vue advice

### JSX
- Migrating React JSX code to Vue or getting attribute type errors → See [render-function-jsx-vue-vs-react](reference/render-function-jsx-vue-vs-react.md)
